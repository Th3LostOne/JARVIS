════════════════════════════════════════════════════════════════════
  MEDIA CONTROLS
════════════════════════════════════════════════════════════════════

  "play"  /  "pause"            Toggle playback
  "next song"  /  "skip"        Next track
  "previous song"               Previous track
  "stop music"                  Stop playback

  Add "spotify" or "music" to target Spotify directly.
  e.g. "pause spotify"  /  "next spotify song"

════════════════════════════════════════════════════════════════════
  VOLUME
════════════════════════════════════════════════════════════════════

  "volume up"                   Increase volume  (stepped)
  "volume down"                 Decrease volume  (stepped)
  "mute"                        Toggle mute
  "set volume to 50"            Set exact level  (0 – 100)
  "set volume to 75 percent"    Same with percent keyword

════════════════════════════════════════════════════════════════════
  AUDIO OUTPUT  —  powered by SoundSwitch  (Alt+F11)
════════════════════════════════════════════════════════════════════

  "switch audio"                Cycle to next output device
  "switch output"               ↑ same
  "cycle audio"                 ↑ same
  "switch to headphones"        ↑ same
  "switch to speakers"          ↑ same

  Requires SoundSwitch running in the system tray.

════════════════════════════════════════════════════════════════════
  SYSTEM TRAY  —  right-click the  J  icon in the taskbar
════════════════════════════════════════════════════════════════════

  Status                 Speaks "all systems operational"
  Screenshot             Takes a screenshot to Desktop
  System Info            Speaks CPU % and RAM %
  Lock Screen            Locks the workstation
  ──────────────────────
  Open Downloads         Opens Downloads folder
  Open Desktop           Opens Desktop folder
  Open Notes             Opens jarvis_notes.txt
  View Log               Opens jarvis.log  (auto-clears every 10 min)
  Restart Ollama         Kills and restarts the Ollama server
  ──────────────────────
  Voice Commands (Help)  Shows this command list as a popup
  ──────────────────────
  Restart JARVIS         Reloads JARVIS instantly
  Quit JARVIS            Exits the assistant
